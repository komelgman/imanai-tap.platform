# Trading Analytics Portal

Утилита/Портал для широкого круга лиц заинтересованных в анализе долгосрочных трендов на крипто-рынке.

## System Context 

![System Landscape Diagram](embed:SystemLandscape)

## MVP

* Фаза 1.
    - Подключаем только одну биржу в качестве источника данных: Binance.
    - Только 10 топовых инструментов.
    - Клиентский интерфейс пока только для показа статистики по инструментам и текущих котировок.
* Фаза 2.
    - Подключить еще минимум 2 источника данных (Gate.io, ByteBit, Polonex).
* Фаза 3.
    - Клиентский UI, показ графиков.
    - Работа с потоками данных:
        - переключение между источниками; 
        - микс источников (для непрерывности истории при листинге/делистинге/релистинге с изменением номинации и названия).
* Фаза 4. 
    - Клиентский UI, рисование примитивов, группировка (Object Tree)
